# Platform Test Automation

A lightweight Python-based system validation project that simulates platform health checks for core system metrics such as CPU usage, memory utilization, disk usage, service health, and latency.

## Features
- Validates platform health metrics from JSON input
- Uses structured logging for execution visibility
- Includes automated unit tests with pytest
- Designed to reflect platform validation and test automation workflows

## Tech Stack
- Python
- Pytest
- JSON
- Logging

## Run the validator
```bash
python validator.py sample_metrics.json
